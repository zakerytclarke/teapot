# 🫖 Teapot LLM

Teapot is a small open-source language model (~800 million parameters) fine-tuned on synthetic data and optimized to run locally on resource-constrained devices such as smartphones, browsers, and CPUs. Built by and for the community, Teapot focuses on grounded reasoning, efficiency, and real-world deployment rather than oversized cloud-only models.

### 🔍 What Teapot Enables
- 📚 Hallucination-resistant Question Answering (Q&A)
- 🔎 Retrieval-Augmented Generation (RAG)
- 🧾 Structured JSON Extraction
- 🏷️ Text Classification & Tagging
- 📊 Semantic Retrieval and Ranking
- 📱 On-Device & Edge AI Applications

Because Teapot models run locally, developers can build AI systems that are:
- Faster (lower latency than large hosted LLMs)
- More private (no external API dependency)
- More cost-efficient (no per-token API costs)
- Easier to deploy in real products and mobile environments
